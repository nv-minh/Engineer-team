# Agents Overview

Complete reference for all EM-Team agents (24 agents total).

## Core Agents (8 agents)
1. **planner** - Create implementation plans
2. **executor** - Execute with atomic commits
3. **code-reviewer** - 5-axis code review
4. **debugger** - Systematic debugging
5. **test-engineer** - Test strategy & generation
6. **security-auditor** - OWASP assessment
7. **ui-auditor** - Visual QA
8. **verifier** - Post-execution verification

## Discovery & Learning Agents (3 agents) - NEW in v1.2.0
9. **market-intelligence** - Market analysis and competitive intelligence
10. **learn** - Project learnings management (patterns, pitfalls, preferences, ADRs)
11. **autoplan** - Multi-phase review pipeline (CEO, Design, Eng, DX)

## Expert Agents (9 agents)
12. **architect** - Architecture & technical design
13. **backend-expert** - Backend specialist (API, DB, performance)
14. **frontend-expert** - Frontend specialist (React, Next.js, UI/UX)
15. **database-expert** - Database specialist (schema, queries, fintech)
16. **product-manager** - Requirements, GAP analysis, market fit
17. **senior-code-reviewer** - 9-axis deep code review
18. **security-reviewer** - OWASP + STRIDE security
19. **staff-engineer** - Root cause analysis, cross-service impact
20. **team-lead** - Team coordination

## Advanced Agents (4 agents)
21. **researcher** - Technical exploration
22. **codebase-mapper** - Architecture analysis
23. **integration-checker** - Cross-phase validation
24. **performance-auditor** - Benchmarking & optimization

**Specialized Orchestrator:**
25. **techlead-orchestrator** - Distributed investigation coordination

**NEW in v1.2.0:** 3 new agents added for discovery & market intelligence (market-intelligence, learn, autoplan). Total agents increased from 22 to 24.

See [Usage Guide](../guides/usage-guide.md#using-agents) for details.
