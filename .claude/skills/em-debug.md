# EM:Debug - Systematic Debugging

Debug issues systematically using scientific method.

## Usage

```
Use the em:debug skill to investigate [bug/issue]
```

## Examples

```
Use the em:debug skill to investigate the login timeout issue
Use the em:debug skill to investigate the memory leak
Use the em:debug skill to investigate the API error
Use the em:debug skill to investigate the database connection failure
```

## What This Does

Invokes the **em-debugger agent** to:
- Apply systematic debugging methodology
- Identify root causes
- Create hypotheses and test them
- Document findings
- Suggest fixes
