# EM:Design-Reviewer - Visual Design Review

Visual design review with 6-pillar UI audit (layout, typography, color, spacing, interaction, content).

## Usage

```
Use the em:design-reviewer skill to [review/audit] [component/page]
```

## Examples

```
Use the em:design-reviewer skill to review the dashboard layout
Use the em:design-reviewer skill to audit the landing page design
Use the em:design-reviewer skill to review the mobile app UI
```

## What This Does

Invokes the **em-design-reviewer agent** to:
- Audit visual design against 6 pillars (layout, typography, color, spacing, interaction, content)
- Identify design inconsistencies and accessibility issues
- Suggest improvements with before/after examples
- Evaluate brand consistency
