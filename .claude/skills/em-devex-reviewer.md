# EM:DevEx-Reviewer - Developer Experience Audit

Developer experience audit with TTHW (Time to Hello World) measurement.

## Usage

```
Use the em:devex-reviewer skill to [audit/measure] [DX aspect]
```

## Examples

```
Use the em:devex-reviewer skill to audit the onboarding experience
Use the em:devex-reviewer skill to measure TTHW for new developers
Use the em:devex-reviewer skill to review the API documentation DX
```

## What This Does

Invokes the **em-devex-reviewer agent** to:
- Measure TTHW (Time to Hello World)
- Evaluate developer documentation quality
- Test getting started flows
- Identify DX friction points
- Score developer experience
